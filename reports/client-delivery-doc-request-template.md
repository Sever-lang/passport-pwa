# Шаблон дозапроса документов у клиента / продавца

Чтобы закрыть оставшиеся вопросы по квартире, нужно запросить у продавца следующие документы:

## Обязательные
- актуальная выписка ЕГРН
- выписка о переходе прав
- правоустанавливающий документ
- паспорт продавца / представителя
- согласие супруга или подтверждение отсутствия брака
- справка о зарегистрированных лицах
- справка об отсутствии задолженности по ЖКУ

## При необходимости
- доверенность
- документы по наследству
- документы по маткапиталу
- разрешение органов опеки
- брачный договор / соглашение о разделе имущества
- документы по перепланировке
