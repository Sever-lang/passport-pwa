# Операционный чек-лист: ежедневная работа по проверке квартиры

## 1. Принять заявку
- [ ] Получен адрес объекта
- [ ] Получен кадастровый номер, если есть
- [ ] Получена ссылка на объявление
- [ ] Понятно, кто продаёт
- [ ] Понятно, на каком этапе клиент
- [ ] Собраны документы, которые уже есть
- [ ] Понятно, что именно тревожит клиента

## 2. Определить формат проверки
- [ ] Экспресс
- [ ] Стандарт
- [ ] Расширенная
- [ ] Зафиксирован ожидаемый результат

## 3. Собрать входные материалы
- [ ] Создана папка по объекту
- [ ] Сохранены документы клиента
- [ ] Сохранена ссылка / скрин объявления
- [ ] Зафиксированы исходные данные

## 4. Провести проверку объекта
- [ ] Идентифицирован объект
- [ ] Проверены базовые характеристики
- [ ] Проверено право собственности
- [ ] Проверены обременения
- [ ] Проверена история переходов права

## 5. Провести проверку продавца
- [ ] Проверен продавец по ФССП
- [ ] Проверен продавец по банкротству
- [ ] Проверены суды
- [ ] Проверены полномочия представителя
- [ ] Проверена доверенность, если есть

## 6. Проверить специальные риски
- [ ] Супруг / совместная собственность
- [ ] Маткапитал
- [ ] Несовершеннолетние
- [ ] Наследство
- [ ] Приватизация
- [ ] Дарение / рента
- [ ] Перепланировка
- [ ] Зарегистрированные лица

## 7. Сформулировать вывод
- [ ] Определён уровень риска
- [ ] Зафиксированы красные флаги
- [ ] Составлен список недостающих документов
- [ ] Подготовлены рекомендации перед авансом
- [ ] Подготовлены рекомендации перед сделкой

## 8. Подготовить выдачу клиенту
- [ ] Подготовлена короткая версия
- [ ] Подготовлен полный отчёт
- [ ] Подготовлено итоговое сообщение
- [ ] При необходимости подготовлен дозапрос документов

## 9. Закрытие кейса
- [ ] Клиенту отправлен результат
- [ ] Зафиксирован статус кейса
- [ ] Материалы аккуратно сохранены
- [ ] Понятен следующий шаг: закрыто / ждём документы / идём в расширение
