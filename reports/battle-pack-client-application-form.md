# Форма заявки на юридическую проверку квартиры

## Короткая форма
Пришлите:
1. адрес квартиры;
2. кадастровый номер, если есть;
3. ссылку на объявление;
4. стоимость объекта;
5. кто продаёт: собственник / представитель / агент;
6. на каком вы этапе: подбор / аванс / сделка;
7. какие документы уже есть;
8. что вас настораживает, если есть сомнения.

---

## Полная форма

### 1. Данные по объекту
- адрес:
- кадастровый номер:
- цена:
- ссылка на объявление:
- тип объекта:

### 2. Данные по продавцу
- кто продаёт:
- ФИО продавца / представителя:
- есть ли доверенность:
- известно ли семейное положение:

### 3. Стадия сделки
- просто рассматриваю объект;
- готовлюсь к просмотру;
- хочу вносить аванс / задаток;
- готовлюсь к сделке.

### 4. Документы на руках
- выписка ЕГРН;
- выписка о переходе прав;
- правоустанавливающий документ;
- паспорт продавца / представителя;
- доверенность;
- согласие супруга;
- справка о зарегистрированных лицах;
- справка по долгам ЖКУ;
- иные документы.

### 5. Особенности, если есть
- наследство;
- приватизация;
- маткапитал;
- дети / опека;
- дарение;
- частые перепродажи;
- перепланировка;
- иные сомнения.

### 6. Что нужно от проверки
- быстрый отсев объекта;
- решение перед авансом;
- полноценный вывод перед сделкой;
- список рисков и документов;
- рекомендации по безопасному сценарию сделки.
