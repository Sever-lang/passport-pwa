# Мини-анкета клиента для проверки квартиры

Чтобы начать проверку, пришлите:

1. адрес квартиры
2. кадастровый номер, если есть
3. ссылку на объявление
4. стоимость объекта
5. кто продаёт: собственник / представитель / агент
6. на каком вы этапе: смотрю варианты / хочу вносить аванс / готовлюсь к сделке
7. какие документы уже есть на руках
8. что именно вас настораживает, если уже есть сомнения

Если есть документы от продавца, сразу приложите всё, что есть.
