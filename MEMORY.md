# MEMORY.md

## Рабочие конвенции по скиллам

- **healthcheck**: использовать для аудита безопасности/харденинга хоста и OpenClaw; любые изменяющие систему действия — только после явного подтверждения пользователя; сначала read-only проверки и план.
- **notion**: работать через Notion API с обязательным заголовком `Notion-Version: 2025-09-03`; помнить различие `database_id` vs `data_source_id`.
- **openai-image-gen**: для пакетной генерации изображений; при запуске через exec закладывать увеличенный timeout.
- **openai-whisper-api**: транскрибация аудио через OpenAI `/v1/audio/transcriptions`.
- **skill-creator**: при создании/обновлении скиллов держать SKILL.md компактным, использовать progressive disclosure, выносить детали в `references/`, код — в `scripts/`, не создавать лишние README/CHANGELOG.
- **weather**: использовать только для текущей погоды и прогнозов через wttr.in/Open-Meteo; не для истории и погодных предупреждений.
- **antidetect-browser**: для браузерных задач использовать только xvfb + Patchright через скрипты скилла; не использовать browser tool; действия на странице — только по ref из `snapshot.mjs`.
- **canva**: работа с Canva Connect API для дизайнов, экспортов, шаблонов и ассетов.
- **google-calendar**: работа с Google Calendar API через env-переменные и Python-обёртку.
- **searxng**: использовать локальный SearXNG для приватного веб-поиска.
- **upx5-image-generation**: если пользователь прислал изображение и просит правку/вариант/дорисовку — всегда использовать `edit.mjs` с `--image`; не заменять это генерацией с нуля.
